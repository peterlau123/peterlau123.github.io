# Peter Lau Blog

Personal blog hosted at [peterlau.me](http://peterlau.me)

## Setup

1. Install [Ruby](https://www.ruby-lang.org/en/) and [Bundler](https://bundler.io/)

2. Install dependencies:

```sh
bundle install
```

3. Run locally:

```sh
bundle exec jekyll serve
```

The site will be available at `localhost:4000`.
