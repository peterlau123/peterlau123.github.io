# 博客重构计划 - Chirpy风格

## 目标
将现有博客渐进式重构为Chirpy风格的简洁、文本优先的设计

## 现有结构分析
- 布局：顶部导航 + 右侧边栏 + 中间内容区
- 样式：Bootstrap框架，头图式设计
- 导航：横向顶部导航栏
- 首页：传统博客列表

## 目标结构（Chirpy风格）
- 布局：左侧固定边栏 + 主内容区
- 样式：浅色系，极简设计
- 导航：侧边栏垂直导航
- 首页：卡片式文章列表
